# What is Secondhand Marketplace

Secondhand Marketplace is a 2nd Year group project for University Of Bristol students spanning an entire academic year as part of the Software Engineering Project module.

The project is a toy marketplace where users can upload their own listings - focusing on a clean UI with gamification elements where users can gain Experience Points (XP)

## What did I do?
- Project Manager for 3/4 of development
- Lead backend designer & programmer with [Flask](https://flask.palletsprojects.com/en/stable/)
- Wrote basic frontend functionality with [Flutter](https://flutter.dev/)
- Lead documentation writer
- Configured database on [Supabase](https://supabase.com/)
- Reviewed peers PRs
